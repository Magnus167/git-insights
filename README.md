# git-insights

A cli tool to generate Git repo stats and insights.

## Features

- [x] Commit count
- [x] Contributor stats
- [ ] Heatmaps/Hotspots
- [ ] Per user stats
- [ ] File-owner estimations
- [ ] Export to CSV/JSON
- [ ] ...

## Installation

Installing from source:

```bash
cargo install --path .
```
